# IGVC Autonomous Ground Vehicle (Senior Design)

Autonomous ground vehicle developed for the Intelligent Ground Vehicle Competition (IGVC) at the University of Texas at Arlington.

## My Role
- Embedded Systems & Sensor Integration
- Worked with TM4C microcontroller and Jetson Orin NX
- Integrated LiDAR, GPS, and camera systems
- Developed real-time control and communication
- Performed system debugging and testing

## Technologies
- Jetson Orin NX
- LiDAR, GPS, Camera
- TM4C Microcontroller
- Embedded C / C++
- I2C, UART, SPI

## Features
- Autonomous navigation
- Sensor fusion
- Real-time control system
